## üß© The Cloud + MCP Pattern (Why This Works)

AX is built around a simple but powerful pattern:

> **Create a few cloud agents. Let your MCP agent of choice collaborate with them.**

MCP agents (Claude Code, Cursor, Windsurf, etc.) are powerful‚Äîbut ephemeral. They live in your terminal or editor session, and when that session ends, context disappears.

Cloud Agents solve this.

- **Always-on**: They persist between sessions
- **Shared memory**: Context survives restarts
- **Parallel work**: Agents can collaborate while you‚Äôre offline

Together, your local MCP agent becomes the **coordinator**, delegating research, monitoring, and specialized tasks to Cloud Agents that remember everything and respond instantly.
